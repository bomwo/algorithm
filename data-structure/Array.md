#Array
Array는 같은 종류의 데이터를 순차적으로 저장하는 자료구조 python 에서는 list 로 구현이 되어 있다.
index를 가지고 있어 직접 접근이 가능하다

장점 : 빠른 접근이 가능하다
단점 : 데이터 추가와 삭제에 비용이 많이 사용된다.
데이터 추가시 공간이 많이필요하면 삭제 시 빈 공간이 생겨 이를 관리 해주어야한다. 길이 조절이 어렵다.


# 1차원 배열
```python3
data = [1,2,3,4,5]
```

# 2차원 배열
```python
data = [[1,2,3],[4,5,6],[7,8,9]]
```
